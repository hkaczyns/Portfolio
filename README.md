# Portfolio
To portfolio zawiera najważniejsze wykonane przeze mnie projekty informatyczne na Politechnice Warszawskiej. W poszczególnych folderach znajdują się ich kody źródłowe, a poniżej krótki opis ich zawartości.

## Foldery
- Folder `Algorytmy_i_Struktury_Danych` zawiera projekty implementujące i badające różne istotne oraz popularne algorytmy w języku Python,

- Folder `Bazy_Danych_2` zawiera projekt aplikacji webowej "Cling Clang" służącej do znajdowania przyjaciół, wykonany w języku TypeScript z wykorzystaniem frameworka React i Express.js. Element bazodanowy został zrealizowany przy użyciu PostgreSQL,

- Folder `Inżynieria_Uczenia_Maszynowego` zawiera projekt modelu sztucznej inteligencji dobierającego odpowiednie ceny dla proponowanych ofert wynajmu wykonany w języku Python,

- Folder `Programowanie_Aplikacyjne_oraz_Bazy_Danych_1` zawiera projekt desktopowej gry strategicznej "Habemus Rex". Jest on napisany w języku Java współpracującym z bazą danych MySQL,

- Folder `Programowanie_Obiektowe` zawiera projekt desktopowej gry fabularnej "Szpady i Zwady" napisany w języku C++,

- Folder `Programowanie_Sieciowe` zawiera projekty implementujące pomniejsze zadania związane z obsługą serwera i klienta w połączeniach UDP oraz TCP w językach C i Python oraz większy projekt implementujący serwer IRC w języku C++,

- Folder `Projekt_Zespołowy_2_Aplikacja_Webowa` zawiera projekt aplikacji webowej do zarzadzania szkołą tańca. Na frontendzie wykorzystuje ReactJS i Typescript, a na backendzie Python z FastAPI. Baza danych to PostgreSQL,

- Folder `Przeszukiwanie_i_Optymalizacja` zawiera projekt implementujący i badający algorytm ewolucyjny w języku Python, który ma za zadanie optymalizację rozmieszczenia sieci optycznej,

- Folder `Uczenie_Maszynowe` zawiera projekt implementujący i badający ulepszony algorytm indukcji reguł IRep++ w języku Python,

- Folder `Wprowadzenie_do_Sztucznej_Inteligencji` zawiera projekty implementujące i badające różne metody zastosowania sztucznej inteligencji w języku Python,
